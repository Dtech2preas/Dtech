Termux Initial Setup

Run these commands in Termux to update your package lists, upgrade installed packages, and install common tools like Python, curl, git, and pip.

pkg update && pkg upgrade -y
pkg install python git curl wget -y
pip install --upgrade pip
pkg install nano vim openssh -y
